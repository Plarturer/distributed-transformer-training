# Distributed Transformer Training

![Version](https://img.shields.io/badge/version-0.1.0-orange) ![Python](https://img.shields.io/badge/python-3.8%2B-blue)

Scalable training framework for Large Language Models. Built for multi-node, multi-GPU environments.

## Installation
```bash
git clone https://github.com/Plarturer/distributed-transformer-training
cd distributed-transformer-training
pip install -e .
```
